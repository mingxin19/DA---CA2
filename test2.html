The Dataset :¶

The Dataset chosen was the annual employment survey from 2008 to 2017 in Ireland. It has the employment amount for different sectors and regions for both full-time and part-time. It also has two columns for the employment growth in percentage from 2016 to 2017 and 2008 to 2017.

The type of data for the dataset values are ratio and interval because the values of employment are whole numbers and it doesn’t have any negative values. But the values for the percentage difference between the years is interval and continuous because some percentage values are negative.

For the project, the aim is to explore into the chosen dataset to see what information could be found. Some questions were raised and hypotheses were formed about the dataset, so the aim of these findings is to help answer these questions and hypotheses.
